# Announcement: Unofficial RWBYM 1.20.1 Forge Port

This repository publishes the source code for an unofficial Minecraft Forge 1.20.1 port of RWBYM.

The port is provided so users and developers can inspect, build, modify, and redistribute the modified source under GPLv3. It is a work in progress and should not be treated as an official RWBYM release.
