# RWBYM 1.20.1 Forge Port

This repository contains an unofficial Minecraft Forge 1.20.1 port of RWBYM.

RWBYM was originally created by the RWBYM authors and published at:

- Original GitHub repository: https://github.com/Blaez-Dev/RWBYM
- Original project/site: https://blaez-dev.github.io/RWBYM/

This port exists to make the mod source available for Minecraft 1.20.1/Forge 47.x users and developers. It is not an official release from the original RWBYM maintainers.

## Status

The port is a work in progress. Weapons, armor, item models, textures, creative tab presentation, and gameplay systems are being migrated and repaired against Minecraft 1.20.1.

Current target:

- Minecraft: 1.20.1
- Forge: 47.x
- Java: 17

## Build

From the repository root:

```bash
./gradlew build
```

On Windows:

```bat
gradlew.bat build
```

The built mod jar is generated under `build/libs/`.
